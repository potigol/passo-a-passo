Como usar

  1. Baixe a versão mais recente do Potigol
  2. Descompacte o arquivo
  3. Para executar o Editor de Código digite no prompt do terminal
    4. java -jar epotigol.jar
  4. Para executar um programa em Potigol digite no prompt do terminal
    5. java -jar potigol.jar arquivo.poti
  • No Windows basta usar epotigol ou potigol arquivo.poti.
